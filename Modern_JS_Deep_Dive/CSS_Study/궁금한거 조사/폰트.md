## HTML 파일에 폰트 적용하는 방법

```html
<head>
    <link href="https://fonts.googleapis.com/css?family=Roboto&display=swap" rel="stylesheet" />
</head>
```

HTML의 head 태그에 위 link 태그를 만들어서 `href` 에 해당 폰트 주소를 작성한다
